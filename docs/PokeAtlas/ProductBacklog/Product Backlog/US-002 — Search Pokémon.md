# US-002 — Search Pokémon

Epic: Collection Tracking
Priority: P0
Status: Not started

**As a** trainer, **I want to** search Pokémon by name or Pokédex number. **So that** I can quickly update my collection.

## Acceptance Criteria

- Trainer can search by Pokémon name.
- Trainer can search by dex number.
- Results update quickly.
- Search works across the full collection.

Example:

```bash
pikachu
025
rookidee
```
