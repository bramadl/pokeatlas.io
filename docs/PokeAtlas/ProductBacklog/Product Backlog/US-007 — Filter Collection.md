# US-007 — Filter Collection

Epic: Collection Tracking
Priority: P1
Status: Not started

**As a** trainer, **I want to** filter Pokémon collection results. **So that** I can focus on a subset of Pokémon.

## Acceptance Criteria

Trainer can filter by:

```bash
Region
Owned
Missing
```

Examples:

```bash
Kanto only
Missing only
Owned only
```
