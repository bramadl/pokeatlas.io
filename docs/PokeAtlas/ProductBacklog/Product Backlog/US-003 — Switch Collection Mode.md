# US-003 — Switch Collection Mode

Epic: Collection Tracking
Priority: P0
Status: Not started

**As a** trainer, **I want to** switch collection tracking modes. **So that** I can track different types of ownership.

## Acceptance Criteria

Trainer can switch between:

```bash
Standard
Shiny
Hundo
Shadow
Purified
```

Current mode affects tracking behavior.

Example:

```bash
Mode = Shiny
Tap Bulbasaur
→ mark shiny
```
