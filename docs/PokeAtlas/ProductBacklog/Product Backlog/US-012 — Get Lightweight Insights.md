# US-012 — Get Lightweight Insights

Epic: Progress & Insights
Priority: P1
Status: Not started

**As a** trainer, **I want to** receive simple progress insights. **So that** I feel momentum in my journey.

## Acceptance Criteria

Insights must be:

- descriptive
- lightweight
- derived

Examples:

```bash
Only 5 Pokémon left in Johto

Shiny completion:
31%
```

System must not:

```bash
recommend teams
coach gameplay
dictate goals
```
