# US-003 — Multi-Brush Collection Tracking

Epic: Collection Tracking
Priority: P0
Status: Not started

**As a** trainer, **I want to** select one or multiple tracking brushes. **So that** I can quickly track collectible targets across many Pokémon.

## Acceptance Criteria

- Trainer can open a tracking brush toolbar.
- Trainer can select multiple brushes:
  - Shiny
  - Hundo
  - Nundo
  - Shadow
  - Purified
  - Lucky
  - Eraser
- Trainer can tap Pokémon cards to apply active brushes.
